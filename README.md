# Research Paper Q&A Agent

A learning project for building a RAG-based agent that can answer questions about
research papers — using PyTorch, LangChain, LangGraph, and FAISS.

## Quick Start

See [`docs/01_plan_of_action.md`](docs/01_plan_of_action.md) for the step-by-step build plan.  
See [`docs/02_learning_concepts.md`](docs/02_learning_concepts.md) to understand each technology before you use it.

## Phases

| Phase | Goal |
|---|---|
| 0 | Environment setup |
| 1 | PDF ingestion pipeline |
| 2 | Basic retrieval chain |
| 3 | LangGraph agent |
| 4 | Polish & CLI |
